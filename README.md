python で bot 書くためのフレームワークです。

## インストール

using pip

```
pip install exmachina
```

using poetry

```
poetry add exmachina
```

## 開発

### init

```bash
poetry install
poetry shell
```

### fmt

```
poe fmt
```

### lint

```
poe lint
```

### test

```
poe test
```
